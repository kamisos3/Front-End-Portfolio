MedPost is a mobile platform that works for iPad, iPhone and Web (landing page only) that allows medical facilities to look for real verified workers in the nursing and caregiver career's; that was created in collaboration. It has semantic-search with the Hugging Face sentence-transformers AI, the home has everything thats active in the platform and navigates back the profile. The front-end was developed using TypeScript, JavaScript, Emotion CSS, Expo Go and Ngrok to connect the back-end's FastAPI implementation allowing the application to be accessed from a mobile iOS device. Please visit https://github.com/Egmarortiz/MedPost to learn more about this project.
